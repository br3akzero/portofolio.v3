Update 2025

It's been a while since I've updated this space, and a lot has changed. Today I'm excited to share the new version of my portfolio and blog.

A Fresh Start

I decided to rebuild everything from scratch. The previous version served me well, but it had become overcomplicated. This time, I wanted something minimal, fast, and easy to maintain.

The Stack

The new site is intentionally simple:

  • Vite for blazing-fast builds and development
  • TypeScript for the little JavaScript that's needed
  • Plain HTML & CSS for the content
  • No frameworks — just static files

No React, no Next.js, no complex build pipelines. Just clean, semantic HTML with some CSS for styling. The result is a site that loads instantly and is trivial to deploy anywhere.

Design Philosophy

I went with a dark theme and glassmorphism-inspired cards. The design is intentionally understated—I want the content to be the focus, not flashy animations or effects. Everything is responsive and works well on any device.

What's Next

This rebuild marks a renewed commitment to sharing my work and writing. Here's what you can expect:

  • More technical deep-dives on iOS, Swift, and native development
  • Tutorials and guides from real-world projects
  • Updates on open-source work and side projects
  • Thoughts on software engineering and the craft of building products

I've migrated all the existing blog posts and cleaned them up. Moving forward, I plan to publish more regularly.

Stay Connected

If you want to follow along, check out my GitHub or YouTube channel. New content is on its way.

Thanks for reading, and welcome to the new site.
